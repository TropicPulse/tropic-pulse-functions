2. CORE PRINCIPLES (AI MUST LEARN THESE)
2.1 Single‑Pass Normalization
Every subsystem snapshot must be normalized once, producing:

normalized core

normalized events

normalized decisions

derived fields

severity

lineage

transitions

signal text

VRAM text

pipeline status

trace extras

Panels must never re‑normalize.

2.2 Single‑Pass Classification
Events must be classified once into:

fallback

drift

heal

other

Panels must never re‑filter.

2.3 Derived Values Belong in Normalized Snapshot
Examples:

severity

VRAM text

signal text

lineage chain

transitions list

pipeline severity

event counts

decision summaries

Panels must never compute these.

2.4 Panels Render From Normalized State
Panels must be pure renderers:

js

renderGpuEvents(normalized.events)
renderGpuDecisions(normalized.decisions)
renderGpuTrace(normalized.trace)
No logic.
No filtering.
No normalization.
No classification.
